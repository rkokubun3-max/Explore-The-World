# 🌍 Explore The World

Web application for exploring beautiful destinations around the world. Built with Node.js, Express, and EJS as part of the Full-Stack Web Development journey.

---

## 🚀 Features
* **Custom Styling**: Fully styled with custom CSS (`main.css`) for a clean and modern look.
* **Clean Structure**: Well-organized project layout separating public assets, views (partials), and server logic.

## 🛠️ Tech Stack
* **Backend:** Node.js, Express.js
* **Frontend:** EJS (Embedded JavaScript templates), CSS3
* **Package Manager:** npm
